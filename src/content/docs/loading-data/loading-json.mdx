JSON (JavaScript Object Notation) is a lightweight data interchange format that is easy for humans to read and write,
and easy for machines to parse and generate.
It is widely used for data storage and communication between web applications and servers.

## Loading JSON Files

To load JSON files in Python, we can use the `pandas` library, which provides a convenient way to read JSON data into DataFrame objects for easy data manipulation and analysis.

export const exampleCode = `import pandas as pd

# Load JSON data into a DataFrame\ndf = pd.read_json('data/data.json')\nprint(df)\n\n# Access specific fields\nfor index, row in df.iterrows():\n\tprint(row['name'], row['age'])`;

## Handling Nested JSON

JSON files can contain nested structures, such as lists and dictionaries.
Pandas can handle nested JSON data, but you may need to normalize the data to flatten it for easier analysis.

export const nestedExample = `import pandas as pd\nfrom pandas import json_normalize

# Load nested JSON data\ndf = pd.read_json('data/nested_data.json')

# Normalize the data\nnormalized_df = json_normalize(df['nested_field'])\nprint(normalized_df)`
